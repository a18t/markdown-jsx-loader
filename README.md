# markdown-jsx-loader

Webpack loader that parses markdown as JSX

## Installation

```sh
$ npm install --save markdown-jsx-loader
```

## Usage

```js
var markdownJsxLoader = require('markdown-jsx-loader');

markdownJsxLoader('Rainbow');
```
